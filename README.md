# Hermes Clipper ☤

**The intelligent, agent-driven web clipper for Obsidian.**

Hermes Clipper offloads the heavy lifting of web research—categorization, summarization, and metadata management—to the [Hermes Agent](https://hermes-agent.nousresearch.com).

## ✨ Features
- **Agent-First:** Automatically organizes notes based on your vault structure.
- **Direct Extraction:** Built-in Markdown extraction for quick clips.
- **Custom Templates:** Fully configurable note layouts.
- **Obsidian Native:** Generates clickable `obsidian://` links.

## 🚀 Quick Start
```bash
# Clone the repository
git clone https://github.com/frostmute/hermes-clipper
cd hermes-clipper

# Install locally
pip install -e .

# Run setup
hermes-clip setup
```

## 🤖 Hermes Integration
Copy the `hermes_skills/clipping` directory to your `~/.hermes/skills/` folder to enable agent-driven clipping.
