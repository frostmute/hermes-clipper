What is this?

A decentralized suite of tools for **intelligent web clipping** to Obsidian. We don't just "clip" text; we dispatch an agent to research, synthesize, and organize your data while you're busy hunting bigger game.

The Suite

🧠 The Brain

The clipping skill for Hermes Agent. Handles research, categorization, and cross-linking.

📡 The Bridge

A FastAPI local server that talks to browsers and plugins. Secure and efficient.

🛰️ The Scout

One-click browser extension (Chrome/Firefox). Zero-config link capture.

🔨 The Hand

The hermes-clip CLI. Powerful, standalone, and built for speed.

🏡 The Garden

The Obsidian plugin. Ribbon icon for 1-click synthesis and status monitoring.

Deployment Options

Run it all locally on your machine, or put the Bridge on a dedicated server to clip from multiple devices. It's your second brain; we just keep the lights on.

hermes-clip serve --host 0.0.0.0 --port 8088
